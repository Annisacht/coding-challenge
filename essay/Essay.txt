1.	React adalah library dari javascript yang digunakan untuk membangun design interface dari sebuah web. React dikembangkan oleh facebook, react dirancang untuk mempermudah pengembangan aplikasi.
2.	Keunggulan react sendiri antara lain:
    - Memudahkan para developer untuk membagi komponen yang bisa diatur ulang. Sehingga kode dapat digunakan kembali (reusable).
    - React menggunakan virtual dom sehingga mengurangi beban pada DOM dan meningkatkan kinerja aplikasi
    - React memiliki komunitas yang kuat dan besar sehingga banyak sumber daya dan dokumentasi yang dapat membantu pemula untuk pengembangan aplikasi.
3.	SPA adalah singkatan dari single page application dimana aplikasi web hanya memuat satu halaman html dan memperbarui kontennya secara dinamis ketika user berinteraksi dengan aplikasi.
4.	State dan props:
    - state adalah objek javascript yang berkaitan dengan komponen dan digunakan untuk menyimpan data lokal dalam komponen
    - props adalah cara untuk berkomunikasi antar komponen
    perbedannya adalah :
    - state digunakan untuk private data dari komponen yang bisa berubah-ubah
    - props digunakan untuk mentransfer data dari satu komponen ke komponen lain.
5.	Contoh studi kasus penggunaan state dan props ada pada aplikasi todo dimana ada daftar tugas dan ceklis tugai sebagai selesai. Komponen ‘task’ memiliki state untuk menghandle status ‘selesai’ pada setiap tugas, sedangkan komponen ‘tasklist’ menerima data tugas melalui props dari parentnya yaitu task.
6.	State management adalah cara aplikasi menyimpan dan mengelola data agar dapat diakses dengan konsisten di seluruh komponen. Penggunaan state management adalah ketika menjalankan seluruh fitur
7.	ORM adalah Object-Relational Mapping tujuannya untuk menghubungkan antara objek dengan kode pemrograman dengan tabel dalam basis data relational. Sedangkan ODM adalah Object-Document Mapping Menghubungkan dan mengintegrasikan antara objek dalam kode pemrograman dengan dokumen-dokumen dalam basis data dokumen, seperti MongoDB.
8.	Basis data sql terletak pada model relasional dan terstruktur sedangkat basis data nosql mengadopsi pendekatan yang fleksibel dan tanpa skema untuk menangani data yang beragam dan dinamis dengan lebih menekankan pada skalabilitas dan kecepatan
